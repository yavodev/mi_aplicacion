# 🎓 Plataforma de Cursos en Línea - Psicología

Bienvenido a la **Plataforma de Cursos en Línea de Psicología**, un sistema web diseñado para ofrecer cursos especializados en el área de la psicología. Esta plataforma permite a los usuarios explorar cursos, agregarlos a un carrito de compras y adquirirlos a través de un sistema de pago seguro.

## 🚀 Características Principales

✅ **Exploración de cursos** - Los usuarios pueden navegar y visualizar cursos disponibles con sus detalles.  
✅ **Carrito de compras** - Sistema para agregar y gestionar cursos antes de realizar la compra.  
✅ **Contador de carrito** - Indicador visual de la cantidad de cursos añadidos, ubicado en la parte superior derecha.  
✅ **Sistema de pagos** - Integración con pasarelas de pago para facilitar la compra de cursos.  
✅ **Gestión de sesiones** - Manejo de usuarios y persistencia de datos con PHP y MySQL.  
✅ **Interfaz responsiva** - Diseño amigable, accesible desde dispositivos móviles y computadoras.  

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML, CSS, JavaScript (con jQuery y Bootstrap)  
- **Backend**: PHP (con MySQL para la base de datos)  
- **Base de Datos**: MySQL (gestión de cursos y usuarios)  
- **Servidor Local**: XAMPP

## 📌 Instalación y Configuración

1️⃣ Clona este repositorio en tu máquina local:
```bash
 git clone https://github.com/tu_usuario/mi_aplicacion.git
```
2️⃣ Configura tu entorno local con XAMPP y asegúrate de tener MySQL y Apache activos.  
3️⃣ Crea la base de datos e importa el archivo `database.sql` en phpMyAdmin.  
4️⃣ Configura `config.php` con los datos de tu base de datos.  
5️⃣ Accede al proyecto en tu navegador desde `http://localhost/nombre_del_proyecto`.
